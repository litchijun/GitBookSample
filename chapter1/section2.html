第二节 Gitbook安装

Gitbook安装

Gitbook是从NMP安装的，命令行：

Note: Windows/Linux 下使用如下命令， 不然命令行下不支持：
$npm install -g gitbook-cli

安装完之后，你可以检验下是否安装成功：

$gitbook -V
CLI version: 2.3.2
GitBook version: 3.2.3

如果你看到了与上面类似的版本信息，则表示你已成功完装上了Gitbook，安装了CLI 后，运行gitbook 命令，gitbook会自动安装。
